# 영화관객수 조회 시스템
## 로그인화면
- 로그인화면에 id/pin 찾는 버튼과 회원가입 버튼 추가


## 메인페이지
- 영화관객수 검색 버튼을 누르면 영화리스트 출력이 됩니다.
- 로그아웃 버튼을 누르면 다시 로그인 화면으로 돌아가게 됩니다.
